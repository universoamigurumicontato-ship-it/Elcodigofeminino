# Página de Destino

Una página de destino con diseño moderno para vídeos.

## Archivos

- `index.html` - Estructura HTML
- `style.css` - Estilos y animaciones
- `script.js` - Funcionalidades JavaScript

## Configuración

1. Abra `index.html` en un navegador
2. El vídeo de VTurb se carga automáticamente
3. Personalice el botón CTA editando el `href` en `index.html`

## Recursos

- Diseño responsivo (móvil y escritorio)
- Contador de visualizadores dinámico
- Botón CTA liberado solo después de ver el vídeo completo
- Fondo con degradado oscuro
- Reproductor VTurb SmartPlayer integrado
- Preload y DNS prefetch para mejor rendimiento

## Personalización

### Cambiar emoji
Edite la línea `<div class="emoji-top">🔞</div>` en `index.html`

### Cambiar texto del botón
Edite el texto en `index.html`:
```html
<a href="#" class="btn-primary">
  Quiero hacerla llegar al orgasmo 🔥
</a>
```

### Cambiar color del botón
Edite el degradado en `style.css`:
```css
background: linear-gradient(180deg, #22c55e, #16a34a);
```
